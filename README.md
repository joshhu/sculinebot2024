## 課程簡介
從頭開始開發一個LINE Bot，並且整合OpenAI API，讓LINE Bot能夠回答使用者的問題

## 課程目標
- 課前準備
- 開發環境安裝及測試
- LINE平台基礎知識
- 帳號申請及金鑰
- 基礎觀念及知識
- 程式碼撰寫及說明
- OpenAI API說明及整合
